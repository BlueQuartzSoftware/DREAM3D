File Conversion Plugin

  The file conversion plugin allows the conversion between several different file formats used to store a labeled volume where each label (or value) usually represents a unique grain number. The types of files that can be converted are:
File Type Extension Input Ouput
VTK Rectilinear Grid .vtk Yes Yes
Ph .ph Yes Yes
Dx .dx Yes Yes
DREAM.3D Voxel File .h5voxel Yes NO
