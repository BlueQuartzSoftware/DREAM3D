RemoveJitter {#removejitter}
=====

## Group (Subgroup) ##
UCSB (UCSB)


## Description ##
Computes alignment as with align by misorientation. After all shifts have been found a line is fit and any 'shear' removed (~first order 'background subtraction' on alignment)
