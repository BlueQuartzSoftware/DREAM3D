Unify Triangle Winding {#unifytrianglewinding}
=====

## Group (Subgroup) ##
UCSB (UCSB)


## Description ##
adjusts triangle winding so that face labels 1 < face labels 2

## Parameters ##

## Required Arrays ##
| Type | Default Name |
|------|--------------|
| Face | SurfaceMeshFaceLabels |

None (a valid surface mesh data container is required)


## Created Arrays ##

None
