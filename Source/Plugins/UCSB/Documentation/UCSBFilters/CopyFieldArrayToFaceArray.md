Create Face Array from Field Array {#copyfieldarraytofacearray}
=====

## Group (Subgroup) ##
UCSB (UCSB)


## Description ##
Copies the values associated with a face id to all the faces belonging to that face id.


## Parameters ##
| Name             | Type |
|------------------|------|
| Field Array Name | String |
