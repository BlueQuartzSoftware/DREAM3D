Find Average Orientations Filter

Description | Options | Required Arrays | Created Arrays| Authors

Description

Data required for this filter consists of cell phases, grain Ids, Quats, and an ensemble that contains the crystal structures types. This filter utilizes the required data to calculate the average Quats and the Field Euler angles.
