Summary

This filter finds the deformation statistics by using data from the cells and the Fields. Cell data needed are grain Ids, grain misorientations, kernel average misorientations, misorientation gradients, nearest neighbor distances, and nearest neighbors. Field data are average Quats, Field phases, Average grain misorientations, poles, Schmid factors, and slip systems appropriate to the crystal system.
The out put can be A deformation statistics file, and/or a VTK file.

  • Schmids
