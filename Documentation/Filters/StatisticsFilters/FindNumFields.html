Summary

This filter determines the number of fields that belong to each ensemble. This is accomplished by iterating through the fields and querying the phase of each field.
Note that currently ensembles are solely defined by phase and no other attribute.

Required Field Data

  • FieldPhases

Created Ensemble Data

  • NumFields
