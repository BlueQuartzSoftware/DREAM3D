Group

IO

Description

This filter writes out a file in column format that is space delimited. Header lines are denoted by the "#" character. The columns are the following:

  1. phi1
  2. Phi
  3. phi2
  4. x Position (Microns)
  5. y Position (Microns)
  6. z Position (Microns)
  7. Grain Id
  8. Phase Id
  9. Symmetry

Some information about the phase is included in the header section of the file in addtion to values for the Origin, Step Size, Dimensions and number of grains in the file.
