Description

This filter Reads from a Dx file. This is a file in a format generated using the OpenDX Source Software based on IBM's Visualization Data Explorer.
An example of the format is show below:
object 1 class gridpositions counts 6 5 4
origin 0 0 0
delta 1 0 0
delta 0 1 0
delta 0 0 1

object 2 class gridconnections counts 6 5 4

object 3 class array type int rank 0 items 60 data follows

end
