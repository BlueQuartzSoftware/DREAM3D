Segment Grains (Misorientation) Filter

Description | Options | Required Arrays | Created Arrays| Authors

Description

This filter segments the grains by grouping neighboring voxels that satisfy the misorientation tolerance, i.e., have misorientation angle less than set by the user. The Cell GrainIds established by this filter are used by the other filters. Arrays that must be availble are Cell CellPhases, GoodVoxels, and Quats. Ensemble array required is CrystalStructures.
