Description | Options | Required Arrays | Created Arrays | Authors

Group

Generic

Description

This filter currently copies the Grain Ids Cell data into a newly created Cell Phases Array. This type of operation might be useful if you have segmented data where each pixel/voxel is a "class" from the segmentation routine (such as the EM/MPM algorithm) and the user would like to use this as an initial starting point for both segmenting the classes into individual grains and still retain the original class information for the pixel/voxel.
