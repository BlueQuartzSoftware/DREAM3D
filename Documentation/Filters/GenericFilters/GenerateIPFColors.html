Generate IPF Colors Filter

Description | Options | Required Arrays | Created Arrays | Authors

Group

Generic

Description

This filter will generate Inverse Pole Figure colors for Cubic or Hexagonal Crystal Structures. The determination is based on the Crystal Structure Ensemble data which must be present for this filter to run. The user can enter the Reference direction and is defaulted to [001].

Options

NameType
X Reference DirectionDouble
Y Reference DirectionDouble
Z Reference DirectionDouble

Required Arrays

TypeNameComment
CellCellEulerAngles
CellCellPhases
EnsembleCrystalStructures

Created Arrays

TypeNameComment
CellCellIPFColorsThe array generated is a 3xN array of 8 bit unsigned integers which match up to the standard Red Green Blue colors from 0 - 255. N is the number of voxels in the volume.
