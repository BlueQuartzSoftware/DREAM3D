Find Field Centroids Filter

Description | Options | Required Arrays | Created Arrays| Authors

Description

This filter calculates the centroid of each Field by determining the average x, y, and z position of all the Cells belonging to a Field.
Note that Fields that intersect the outer surfaces of the sample will still have centroids calculated, but they will be centroids of the truncated Field.
