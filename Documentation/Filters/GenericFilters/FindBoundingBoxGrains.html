Find Biased Grains (Bounding Box) Filter

Description | Options | Required Arrays | Created Arrays| Authors

Description

This filter determines which grains are biased by the outer surfaces of the sample and thus excluded from any statistical analysis to be stored at the ensemble data level.
The largest box that does not include any of the centroids belonging to the fields that intersect an outer surface of the sample is first determined. Then each field's centroid is checked to see whether it lies within the box. If the centroid lies within the box, then the field is said to be unbiased and if it lies outside the box, then the field is said to be biased.
It should be noted that fields that do not actually intersect the outer surfaces may still be considered biased (generally small fields) due to the higher probability of larger fields to intersect the outer surfaces.
