Clear Data {#cleardata}
======

## Group (Subgroup) ##
Processing Filters

## Description ##
This filter writes Zeros (0) into every element of every **Cell** Array.


## Parameters ##

| Name | Type |
|------|------|
| X Min | Integer |
| Y Min | Integer |
| Z Min | Integer |
| X Max | Integer |
| Y Max | Integer |
| Z Max | Integer |
| Renumber Grains | Boolean (On or Off) |

## Required Arrays ##
None



## Created Arrays ##
None
