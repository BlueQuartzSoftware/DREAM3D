Change Resolution {#changeresolution}
======

## Group (Subgroup) ##
Sampling Filters (Resolution)

## Description ##
This Filter changes the **Cell** resolution based on inputs from the user for x, y, and/or z values.


## Parameters ##

| Name | Type |
|------|------|
| X Res | Double |
| Y Res | Double |
| Z Res | Double |

## Required Arrays ##
None

## Created Arrays ##
None
