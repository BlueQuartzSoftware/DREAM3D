Write Field Data as CSV File {#fielddatacsvwriter}
======

## Group/Subgroup ##
I/O Filters


## Description ##
This Filter writes the __Field__ data, using a filename input by the user, to an output file in _CSV_ format.


## Parameters ## 

| Name | Type |
|------|------|
| Output File | File Path |

## Required Arrays ##
None

## Created Arrays ##
None
