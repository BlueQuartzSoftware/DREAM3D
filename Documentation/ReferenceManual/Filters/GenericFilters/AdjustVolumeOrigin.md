Adjust Voxel Origin  {#adjustvolumeorigin}
==============================

## Group (Subgroup) ##
Generic Filters (Spatial)

## Description ##
This filter changes the origin  of the volume ***without*** "resampling" the data.

## Parameters ##

| Name | Type |
|------|------|
| X Origin | Float |
| Y Origin | Float |
| Z Origin | Float |

## Required DataContainers ##
Voxel

## Required Arrays ##
None

## Created Arrays ##
None
