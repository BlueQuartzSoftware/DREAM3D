Rename Feature Array {#renamefeaturearray}
==============================

## Group (Subgroup) ##
Generic Filters (Memory/Management)

## Description ##
This filter renames a user chosen **Feature** array to a user defined new name.

## Parameters ##

| Name | Type |
|------|------|
| Feature Array Name | String |

## Required DataContainers ##
Voxel

## Required Arrays ##

| Type | Default Name | Description | Comment | Filters Known to Create Data |
|------|--------------|-------------|---------|-----|
| Feature | *User Chosen* |  | Array will always be present, since the user may only chose from existing arrays | N/A |


## Created Arrays ##

| Type | Default Name | Description | Comment |
|------|--------------|-------------|---------|
| Feature | *User Defined* | | Array will have same values as its old counterpart |
